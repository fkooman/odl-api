# Introduction
This is software for using the ODL API.

## Requirements
### Ubuntu
Install [Composer](https://getcomposer.org).

### Fedora
Install Composer:

    $ sudo dnf -y install composer git

# Installation

	$ git clone https://github.com/fkooman/odl-api.git
	$ cd odl-api
	$ composer install

# Running
	
	$ php f1.php
